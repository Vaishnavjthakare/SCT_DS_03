# SCT_ML_03 - 

This project is part of my Data Science internship tasks at SkillCraft Technology.

1. Task Objective
Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on their demographic and behavioral data using the Bank Marketing dataset.

2. Dataset
The dataset used in this project is the **Bank Marketing Dataset** from the UCI Machine Learning Repository.

Dataset Source:  
https://archive.ics.uci.edu/ml/datasets/Bank+Marketing

Alternative Source used in code:  
https://raw.githubusercontent.com/selva86/datasets/master/bank-full.csv

3. Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
